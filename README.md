# About
Fumo Rescuer - or simply Fumo Recu, is a 2D top-down game where the player have to navigate through many levels to rescue the Mint Fumo. With characters swap/charge system, various levels and mechanics, and upgrades that change the gameplay in many different ways, the game is designed to be challenging, heavily encourages experiment and offers good replayablity :)
